# Verifying on SuiVision

1.  **Copy the Package ID** from the terminal output.
2.  Go to [SuiVision](https://suivision.xyz) (ensure you switch to Testnet/Mainnet).
3.  Paste the Package ID in the search bar.
4.  Navigate to the **'Modules'** tab.
5.  You should see `vault` listed.
6.  Click on the **'Bytecode'** tab to verify that the source matches (Sui supports source verification uploads).
